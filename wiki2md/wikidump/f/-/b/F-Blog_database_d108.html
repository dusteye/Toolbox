F-Blog/database

Wikipedia，自由的百科全书

关于数据库的讨论意见


1.users 用户表--用户的基本信息 　 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES id
name VARCHAR 50 YES 登陆名
password VARCHAR 64 YES password
email VARCHAR 100 YES email
reg_datetime DATATIME 0 注册时间
nickname VARCHAR 50 笔名
url VARCHAR 150 固定链接
       
　 　 　 　 　
2.blogs 记录blog信息，对应与user 　 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES 　 　 　
user_id BIGINT 20 YES 　 　 　
name VARCHAR 50 name 　 　 　
description VARCHAR 255 Blog的描述，说明 　 　 　
visited_num INTEGER 11 被访问次数 　 　 　
comment_num INTEGER 11 评论数 　 　 　
article_num INTEGER 11 发表文章数 　 　 　
css 所选用的皮肤模版▲是地址还是选项　　　
announce TEXT - 公告 　 　 　
3.articles 文章表 　 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES id
title TEXT 65535 标题
content LONGTEXT -1 内容
cat_id BIGINT 20 分类
datetime DATETIME - 最后发布时间
comment_num INTEGER 11 评论数
visit_num INTEGER 11 访问次数
brief TEXT 11 简介▲
user_id BIGINT 20 author ID
       
4.tags tag的基本信息表-----important 　 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES 一定很大
name VARCHAR 50 YES
num INTEGER 11 使用次数
5.groups group表 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES
name VARCHAR 50 YES 群名称
description VARCHAR 255 群描述
user_id BIGINT 20 YES 创立者user_id
datetime DATETIME - 成立时间
url VARCHAR 150 群的访问链接，静态地址
commend_user_id BIGINT 20 推荐blog的user_id
commend_article_id BIGINT 20 推荐文章id
visited_num INTEGER 11 群被访问次数
article_num INTEGER 11 群文章数
member_num INTEGER 11 群成员数
css 群模版
announce TEXT 65535 群公告
6.users_groups 用户与Group的关系表 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES
user_id BIGINT 20 YES
gro_id BIGINT 20 YES
status ENUM - 用来记录状态信息√0表示用户已是群组成员;1表示用户正在申请加入群组;2表示用户被群组拒绝　3表示是群主
7.groups_tags Group设定的Tags关系表 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES
gro_id BIGINT 20 YES
tag_id BIGINT 20 YES
8.articles_tags ARTICLE设定的Tags关系表 　 　 　 　 　
field type size not-null remark
art_id BIGINT 20 YES
tag_id BIGINT 20 YES
9.groups_articles G和A的关系表 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES
gro_id BIGINT 20 YES
art_id BIGINT 20 YES
tag_id BIGINT 20 因为某个tag而加入
10.comments fblog的管理者admin 　 　 　 　 　
field type size not-null remark
com_id BIGINT 20 YES
art_id BIGINT 20 YES
com_name VARCHAR 50 评论人名称(Guest)
com_title TEXT 65535
com_content TEXT 65535
com_email VARCHAR 100 评论人Email
com_url VARCHAR 150 评论人网址，blog地址
com_datetime DATETIME - 评论时间
user_id▲ BIGINT 20 如果是注册用户，则有此项▲可以略去
11.catalogs user的文章分类catalogs 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES id
user_id BIGINT 20 YES 对应的用户名
name VARCHAR YES name
description LONGTEXT -1 描述
12.user_links 用户的link的记录表 　 　 　 　 　
field type size not-null remark
link_id BIGINT 20 YES
name VARCHAR 50 YES
url VARCHAR 150 YES
description VARCHAR 255
user_id BIGINT 20 YES
13.gro_links Group的link的记录表 　 　 　 　 　
field type size not-null remark
id BIGINT 20 YES
name VARCHAR 50 YES
url VARCHAR 150 YES
description VARCHAR 255
gro_id BIGINT 20 YES
