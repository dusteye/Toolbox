windows服务名称：FirebirdServerDefaultInstance

启动服务：net start FirebirdServerDefaultInstance
停止服务：net stop FirebirdServerDefaultInstance
[编辑]

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services

start = {0,1,2,3,4} 

0 == 由核心装载器装载
1 == 由I/O子系统装载
2 == 自动启动
3 == 手工启动
4 == 禁止启动
[编辑]

/bin

gsec.exe 管理数据库username password

isql.exe 相当于mysql.exe，进行数据库操作

[编辑]

INF

INF:Device INFormation File

[编辑]

基本格式

[节点]
key="value1,value2,..."
key="value"
[编辑]

特点

  • 大小写不敏感
  • ;表示注释
  • 节点名最长为255
  • 不需重起就可以添加服务
[编辑]

学习资料

  • INF文档小结
  • INF文档解析
  • 手动利用INF添加系统服务 
[Version]
Signature="$WINDOWS NT$"
[DefaultInstall.Services]
AddService=inetsvr,,My_AddService_Name
[My_AddService_Name]
DisplayName=Windows Internet Service
Description=提供对 Internet 信息服务管理的支持。
ServiceType=0x10
StartType=2
ErrorControl=0
ServiceBinary=inetsvr.exe
