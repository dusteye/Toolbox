Wikipedia，自由的百科全书

[编辑]

proxychains配置

  • 安装proxychains 
#apt-get install proxychains
  • 配置/etc/proxychains.conf 
strict_chain
tcp_read_time_out 15000
tcp_connect_time_out 10000
[ProxyList]
socks5 219.147.187.186 6460
http 219.217.250.3 3128
#socks5 219.147.187.186 6460 proxy ibmtc #支持用户名密码验证
  • 使用proxychains 
#proxychains lynx http://www.kernel.org
