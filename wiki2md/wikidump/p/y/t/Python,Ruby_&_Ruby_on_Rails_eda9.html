Python,Ruby & Ruby on Rails

Wikipedia，自由的百科全书

1.Python是什么？

Python是一种开放源代码的脚本编程语言，这种脚本语言特别强调开发速度和代码的清晰程度。它可以用来开发各种程序，从简单的脚本任务到复杂的、面向对象的应用程序都有大显身手的地方。Python还被当作一种入门程序员最适合掌握的优秀语言，因为它免费、面向对象、扩展性强同时执行严格的编码标准。

Python的最大的特点之一就是他是一个动态语言，很灵活，非常适合用于快速开发。同时有着丰富的类库支持，可以做很多的事情。如果经过编译（是的，可以编译），速度和C/C++写的程序效率上是没有区别的。

Python的官方网站是http://www.python.org

有一个中文社区：http://python.cn/，但是感觉不是特别活跃。


2.Ruby是什么？

Ruby是一种功能强大的面向对象的脚本语言,她可以使您方便快捷地进行面向对象编程.Ruby使文本处理和系统管理变得简单，与Perl非常相似.一句话：简单明了，扩展性强，移植性好.

Ruby的几大亮点: 

语法简单
垃圾回收
可移植性
动态载入
完全免费
正则表达式

Ruby的官方站点是：http://ruby-lang.org/en/

Ruby的中文站点是：http://rubycn.ce-lab.net/


3.Ruby on Rails是什么？

Ruby on Rails是一个 Web 应用程序框架,是一个相对较新的 Web 应用程序框架，构建在 Ruby 语言之上。它被宣传为现有企业框架的一个替代，而它的目标，简而言之，就是让生活，至少是 Web 开发方面的生活，变得更轻松。

Ruby on Rails的官方网站是：http://www.rubyonrails.org

另外，关于Ruby on Rails有一些比较深入的讨论在JavaEye的RoR/Python/Groovy版里面有比较详细的讨论，访问地址是：http://forum.javaeye.com/viewforum.php?f=30


4.Python有关的资料和书籍

关于Python的学习，网上推荐较多的是下面两本书： 

Learning Python，中文版叫《学习Python》 
Dive into Python

另外还有O’Reilly出的两本书，我觉得作为速查手册很不错： 

Python Cook Book 
Python Standard Library

Learning Python以及后两本书都有电子版，chm格式。


5. Ruby和RoR有关的资料

目前没有特别好的Ruby中文资料，我记得去年有一期程序员上有一篇介绍性的文章。

另外，英文资料有两本书： 

Programming Ruby 
Ruby Developers Guide

据说把Ruby的基本语法看了之后就可以开始使用Ruby on Rails进行编程了。关于Ruby on Rails也有2本书： 

Rails4Days 
Agile Web Development with Rails

另外，关于Ruby on Rails的学习，可以看ONLamp.com上的“Rolling with ruby on rails”，也可以看Ruby on Rails官方站点上的Tutorial。

前面一本是一步一步介绍操作的，好像是4天学会Ruby on Rails一类的。后面一本是讲述使用Rails进行敏捷编程，最近这本书很受推崇。 这几本书我都找到了电子版。

6. 关于安装

Python的安装

Python目前最新的版本是2.4.3，到官方站点http://www.python.org的Download目录中下载即可。 校内可以到 

ftp://202.118.224.241/software/Development/Language/python/Python2.4(final)/

下载


Ruby和Ruby on Rails的安装

首先安装Ruby，然后安装Ruby Gem，这是一个类似于apt的包管理的工具。接下来就可以安装Ruby on Rails了。RoR的网站上有比较详细的介绍，不过这种方法对于教育网有点不好用，因为RubyForge在国外，如果实验室作了限制就连不上，因此，还是使用RubyonRails.org上提供的InstantRails包比较合适，这个包中包括了ruby、rails、apache以及Mysql，我的感觉是使用这个最好了，不用装ruby，而且东西很全。
