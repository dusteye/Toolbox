Wikipedia，自由的百科全书

1. 2.6内核的initrd解压修改问题：

综述：使用file查看序性，然后用cpio将其导入目录。 

   debian:/media/usb/boot# cp lookou-20060501-2.img /home/wangyao/
   debian:~# cd /home/wangyao/
   debian:/home/wangyao# mv lookou-20060501-2.img lookou-20060501-2.gz
   debian:/home/wangyao# gzip -d lookou-20060501-2.gz
   debian:/home/wangyao# mkdir lookou
   debian:/home/wangyao# cd lookou
   debian:/home/wangyao/lookou# cpio -i < ../lookou-20060501-2
   4913 blocks
   debian:/home/wangyao/lookou# ls
   bin  dev  etc  init  lib  loopfs  proc  sbin  sys  sysroot

假设当前目录位于准备好的initrd文件系统的根目录下
bash# find . | cpio -c -o > ../initrd.img
bash# gzip ../initrd.img

2. U盘grub建立 

   进入usb根目录
   1.mkdir -p boot/grub
   2.cd boot/grub
   3.cp /boot/grub/{stage1，stage2，grub.conf｝ ./
   4.cd 
   5.umount /dev/sda1 --假设你的U盘是sda1分区
   6.grub
   7.grub> root (hd1,0) --如果你的系统只有1块硬盘，系统将U盘认为是hd1。序号依此类推。
   8.grub> setup (hd1) --与上面的盘符相匹配。
   如此系统的grub建立成功。

3. 软盘grub类似上面的建立过程，只是设备部分不同而已

4. 制作USB的initrd文件 

  mkinitrd --preload=ehci-hcd --preload=uhci-hcd --preload=ohci-hcd --preload=usb-storage \      
  --preload=scsi_mod --preload=sd_mod /boot/lookou6-20060501-1.img 2.6.12

5. 将U盘做一个镜像。 

  dd if=/dev/sda1 of=/home/wangyao/Image/usb-20060506
  卸载掉U盘：
  umount /dev/sda1
  将镜像挂载到系统上。
  mount -o loop /home/wangyao/Image/usb-20060506 /media/usb
