项目概述

该应用是一个基于位置的慈善服务，用户可以通过该应用了解到身边需要帮助的人(或需要救助的动物)，然后施以援手。直接帮助身边的人，让慈善活动更透明、更真实。

[编辑]

应用示例

  1. 如果您想捐赠一些书籍或者资助一个贫困学生，但是不确定具体的对象，你可以使用“身边”了解离您周边需要帮助的希望小学或者贫困学生的信息，然后去实地了解待捐赠(或者资助)对象的真实情况，然后进行直接帮助。
  2. 如果您是一个热爱小动物的人，了解一些流浪猫、流浪狗的信息，希望更多的人为他们提供帮助，您可以将信息提交到该应用中。
[编辑]

UI 元素

  1. 地图
  2. 用户所在位置
  3. 位置(待帮助的人或动物)的marker，marker以不同颜色对类别进行区分
  4. 搜索对话框：包括一个文本输入框（输入关键字），多个复选框（选择慈善活动的类别），确定及取消按钮
  5. 提交慈善信息对话框：参考腾讯微薄以及Foursquare的设计
[编辑]

主要场景截屏

  1. 系统启动画面：体现“身边”、“慈善”的元素
  2. 用户注册界面：iPhone和Android有区别，Android可以直接用系统绑定的Gmail账户注册
  3. 用户登录系统后的缺省界面：显示地图，以及周边的信息。（鉴于信息不会很密集，地图的level无需太高)
  4. 提交慈善信息的界面：可以提交照片和文字信息
  5. 查看慈善信息的界面：查看慈善信息（包括文字，图像预览，签到数）
  6. 在进行慈善活动后签到(Sign in):参考foursqaure等签到设计
