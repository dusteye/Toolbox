用户管理

  1. 用“usermod -G wheel 用户名”将一个用户添加到wheel组中
  2. 修改 /etc/pam.d/su 文件，找到“#auth required /lib/security/$ISA/pam_wheel.so use_uid ”这一行，将行首的“#”去掉
  3. 修改 /etc/login.defs 文件，在最后一行增加“SU_WHEEL_ONLY yes”语句。

系统外接的存储管理方法如下:

  1. 确认存储的设备号，本例中为/dev/sdb
  2. mkfs.ext3 -F /dev/sdb
  3. mount /dev/sdb /srv

远程监控 nagios

安装基础软件包
yum -y install httpd
yum -y install gcc
yum -y install gd gd-devel

创建一个nagios user并为其设置password
useradd nagios
passwd nagios

创建用户组并将nagios user和apache user加入到用户组中
groupadd nagcmd
usermod -a -G nagcmd nagios
usermod -a -G nagcmd apache

下载nagios软件包及nagios插件
wget http://prdownloads.sourceforge.net/sourceforge/nagios/nagios-3.2.3.tar.gz    



编译安装
tar xzf xxx.tar.gz
cd xxx
./configure --with-command-group=nagcmd
make all
make install
make install-init
make install-config



service nagios start
service httpd restart

系统备份脚本

需求概述：完成一个通用备份脚本，可以从配置文件或者命令行读取待备份的目录或者文件 

1 对需备份的文件和目录进行归档并压缩，使用用tar压缩为 tar.gz即可
2 可以从文件读取需要进行备份的目录和文件，将配置文件命名为 backup.list
3 支持的配置选项
  -c 读取配置文件
  -d 备份存放的位置（缺省位置写在程序中）
  -s 需备份的源文件或者目录
  -l 日志的路径(缺省路径为/var/log/backup.log,硬编码在脚本中)
4 备份文件命名规则 文件+时间戳 ，例如 file20110308.tar.gz
5 每个文件备份是否完成，需要写入日志文件
6 上传到指定的FTP(需要使用expect)
