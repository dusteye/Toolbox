测试记录数量：110010

语句：select name from test where name like '%li%';
结果：88008 rows in set (0.48 sec)
说明：按照网上的说法，这种查询是不会调用索引的查询。

语句：select name from test where age < 15;
结果：22019 rows in set (0.11 sec)
说明：这种查询是利用age字段的索引进行的查询。

语句：select name from test where 4*(age-3)<48;
结果：22019 rows in set (0.13 sec)
说明：按照网上的说法，这种查询是不会调用索引的查询。
