只言片语开发经

Wikipedia，自由的百科全书

  1. query.list.get(0)在使用时前应该检查list是否为空，如果list.size()==0，则不能用get(0)方法。
  2. Query.list()的结果处理。这个方法如果查询结果为空List，不是返回null。返回一个size为0的list是个不错的主意。可以避免在使用for(:)的时候出现空指针错误。
  3. 如果你想要引入排序功能，那么就最好不要把排好序的List再放入Map中，因为那样只会让你挠头。当然你可以使用一个特殊的TreeMap。
  4. Map的keySet()、valueSet()只是返回一个Map.Entry的Iteratr，没有新建Set对象。所以在循环过程中不能增删成员。
  5. Java Modifier

AbstractMap源代码中的一段：这两个Modifier只能用来修饰成员 

transient volatile Set<K>        keySet = null;
transient volatile Collection<V> values = null;

transient:成员不可以被序列化

volatile:贡献成员，用于并发线程的共享（？存疑）
