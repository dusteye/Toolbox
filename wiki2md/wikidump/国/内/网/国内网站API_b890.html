Yupoo

分类:图片

URL:http://www.yupoo.com/apidoc/

方式:REST/XML-RPC

开发包:PHP/.NET/C++/Flash AS(网友开发)

简介:

要调用Yupoo的API，您必需先申请并配置一个API Key。

然后您需要选择一种请求形式。目前，Yupoo支持两种请求形式(REST和XML-RPC，它们都基于HTTP协议)。发送一个HTTP请求到特定的地址，并指定调用的方法名和参数，然后会返回格式化的结果.

所有请求都必需以UTF-8编码发送

BaBaLian巴巴变

分类:图片

URL:http://www.bababian.com/api/APIDocument.htm

方式:XML-RPC

开发包:PHP/ASP(网友开发)

简介: 巴巴变的API使用的是UTF-8的编码格式，所有请求和返回的XML字符串都是UTF-8。 需要api_key。可调用方法数量明显逊色于yupoo

图盟

分类:地图

URL:http://code.mapabc.com/

方式:JavaScript封装

开发包:javascript

简介: MapABC地图API为您提供了值得信赖的地图数据，包括全国331个城市的地图和POI（兴趣点），而且这些数据还在不断增加中；MapABC地图 API使用成熟可靠的技术为合作伙伴创造了巨大的价值，也能为您提供非凡的互联网地图体验；MapABC地图API提供了丰富的地图及搜索功能，适应个性化的互联网地图应用需求。

他的合作用户包括谷歌、微软、新浪、Tom、搜房等网站，本身不能算是一个很严格意义上的web2.0网站，而更像是一个针对企业级用户的内容服务提供商。api_key的申请需要注册网站帐号，绑定使用的网址，立即生成key。

51地图（灵图）

分类:地图

URL:http://api.51ditu.com/docs/index.html

方式:iframe/JavaScript/Http+Xml/WebService

开发包:

简介: 该地图应用接口提供免费的基本服务和部分收费服务，在引擎之上，我们提供了iframe、JavaScript、Http+Xml、WebService四种形式的接口，可以满足网站类(B/S结构)和非网站类(C/S结构)的应用。

51地图是北京灵图软件技术有限公司的下属项目，使用的是灵图的技术。它将接口划分成多个大类，如地图频道、地图、标注、搜索、导航、公交、位置描述几大类，不同的接口类提供不完全一样的接口交互方法。提供chm版的API文档下载。但是服务分为免费和收费两种，部分文档公开，高级服务收费。
