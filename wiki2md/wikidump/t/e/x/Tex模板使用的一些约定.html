代码片段

    1. 所有的代码都处于listings环境中，包括头文件说明及作者的代码示例
    2. 所有的代码输出都处于verbatim环境中，包括标准输出和输出文件
    3. 代码缩进: 所有代码都要手动缩进，缩进为2空格
    • example1 输出 
\begin{verbatim}
  cpu=0 is set
  cpu=1 is set
  cpu=2 is unset
  cpu=3 is unset
  ...
  cpu=1023 is unset
\end{verbatim}
    • example2 代码 
\begin{lstlisting}
  cpu_set_t set;
  int ret, i;
  CPU_ZERO (&set);
  ret = sched_getaffinity (0, sizeof   (cpu_set_t), &set);
  if (ret == -1)
     perror ("sched_getaffinity");
  for (i = 0; i< CPU_SETSIZE; i++) {
     int cpu;
     cpu = CPU_ISSET (i, &set);
     printf ("cpu=%i is %s\n", i, cpu ? "set" : "unset" );
  }
\end{lstlisting}

对于error number的列举

\begin{eqlist*}
\item[\textbf{EACCESS}] 进程没有CAP\_SYS\_NICE能力，却企图提高进程优先级。(仅适用 于setpriority())
\item[\textbf{EINVAL}] "which"的值不在PRIO\_PROCESS, PRIO\_PGRP或PRIO\_USER之 中。
\item[\textbf{EPERM}] 指定的进程有效用户ID和调用进程有效用户ID不一致，且调用进程没有CAP\_SYS\_NICE能力。(仅适用于setpriority())
\item[\textbf{ESRCH}] 不存在符合“which”和“who”的进程。
\end{eqlist*}
