浮点数的比较

Question: (float)1.0 与 (float)0.9999999 比较大小的结果，以及(float)1.0 与(float)0.99999999 (9的位数大于等于8)



Answer:由于在IEEE中可以表示的数离散，因此其它数均由与其最接近的数进行表示。
       以C中float类型为例，1.0的编码为0 01111111 00000000000000000000000
       而IEEE可表示的最大小数为0 01111110 11111111111111111111111经计算可知，该数为
       1-2^(-24)，两者的中间数为1-2^(-25)，因此在(1-2^(-25),1）之间的数就应用1.0来表示
       而0.99999999(8个9或8个以上)就介于其中，因此在计算机表示中就表示成了1.0。
       因此，当9的位数小于8是与1.0不等，而位数大于等于8时便与1.0相等了。

通过位运算提高运算速度

http://acm.hit.edu.cn/ojs/show.php?Proid=2013&Contestid=0

题目大意：给定一个m*n的表格G，每个表格有两种状态(1白，0黑)。每次操作可以改变一个格子和它周围的格子的状态(0->1, 1->0, 1白，0黑)。求解所有能够把棋盘变为全黑的方法。

算法：

  1. 状态的表示：　第i行的状态用s[i]表示：s[i]的第j位置为G[i][j]
  2. 一旦第一行的操作确定下来后，以后的每一行都可以通过上一行的状态来确定需要的的操作。如果前一行的j列1,则当前行的第j列必须改变状态，因而周围的方格也得改变状态。改变s[i]相当于s[i] ^= s[i-1], s[i]^=s[i-1]<<1, s[i]^= s[i-1]>>1,第i行操作对i+1行的改变相当于s[i+1] ^= s[i-1].最后将得到最后一行的状态，判断是否全0。

附代码： 

#include <iostream>
const int N = 16;
using namespace std;
int state[N], n, cpy[N], bf[N];
bool search(int p) {
    for(int i=0; i<n; i++) cpy[i] = state[i];
    bf[0] = p;
    cpy[0] ^= p; cpy[0] ^= p>>1;
    cpy[0] ^= (p<<1&((1<<n)-1));
    if(n>1) cpy[1] ^= p;

    for(int i=1; i<n; i++) {
        bf[i] = cpy[i-1];
        cpy[i] ^= cpy[i-1];
        if(i<n-1) cpy[i+1] ^= cpy[i-1];
        cpy[i] ^= ((cpy[i-1]<<1)&((1<<n)-1));
        cpy[i] ^= cpy[i-1]>>1;
    }
    return cpy[n-1] == 0;
}
void print() {
    for(int i=0; i<n; i++)
        for(int j=n-1; j>=0; j--) {
            if(bf[i] & (1<<j)) printf("1");
            else printf("0");
            if(j == 0) printf("\n");
            else printf(" ");
        }
}
int main()
{
    int t = 0;
    while(scanf("%d", &n) ==1) {
        printf("PUZZLE #%d\n", ++t);
        memset(state, 0, sizeof(state));
        int num;
        for(int i=0; i<n; i++)
            for(int j=n-1; j>=0; j--) {
                scanf("%d", &num);
                if(num) state[i] |= (1<<j);
            }
        int maxn = (1<<n), cnt = 0;
        for(int i=0; i<maxn; i++) {
            memset(bf, 0, sizeof(bf));
            if(search(i)) {
                printf("solution #%d\n", ++cnt);
                print();
            }
        }
        if(cnt == 0) printf("impossible\n");
    }
    return 0;
}
