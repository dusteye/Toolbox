522室无线网络管理

[编辑]

使用规则

  1. 请大家在使用无线网络的时候，设置IP获取方式为DHCP
  2. 请大家不要泄漏无线网络的登录密码，违者严惩
[编辑]

基本信息

  • 型号:DLink DWL-2000AP+A
  • IP: 172.16.0.50
  • 存放地点：新技术楼522
  • SSID: FoOTOo ，即查询网络时显示的名称。
  • 系统管理员：吴晋
[编辑]

使用方法

  1. 设置好本机的无线网卡
  2. 请使用DHCP,网关设置为 172.16.0.46,DNS为202.118.224.101
  3. 搜寻无线网络连接，找到FoOTOo
  4. 登录密码：1a2b3c4d5e
  • 管理员用户：admin
  • 管理员密码：ibmtc
