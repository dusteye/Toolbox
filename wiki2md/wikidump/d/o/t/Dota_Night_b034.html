活动5WH

时间：周末下午晚上
地点：各自的座位上
人物：实验室任何留守人员


[编辑]

活动方法

  • 登录46，启动服务 
#~/ghost++/ghost++ default.cfg
  • 进入War3，选择局域网游戏，进入默认游戏，在聊天窗口输入： 
!password 123                            // 这是管理员密码，输入后权限变更为游戏管理员
!pub AnyNameYouLike              // 建立新游戏，默认地图DotA
  • 退出默认游戏，进入建立的新游戏 
!start 5                                          // 开始游戏了！  其他命令自行查阅ghost++相关文档
!map xxx                                     //  更换游戏地图
[编辑]

修改新建立游戏默认DotA版本

  • 在ghost++/mapscfg下修改dota.cfg对应内容
