此文档用以定义Lightning(Flash plugin for eclipse)的详细功能，用以作开发时的参照。
  • 粗体部分是未加入稳定版本的功能。

    • 1. 项目目标定义 本插件的目的是要建立一个开源的flash动画编辑软件，能够在Eclipse中自由的创建动画（小型片断），然后导出成为swf文件。同时，整合目前flash开源社区中所使用的一些开源工具，比如mtasc、swfmill、asdt等工具，打造一个开源的、Linux平台下可使用的Flash创作工具。

    整个面板是一个视图（perspective），分为四个部分，都暂以视图（view）的方式实现。其中workarea和workarea1应以编辑器（editor）的方式在后期实现。

    2. 详细功能定义

    2.1. Tool

    功能名称

    功能详细描述

    相关操作描述

    画矩形

    使用绘图工具在画布上绘制矩形，可以随意确定矩形的长、宽，并可以确定矩形的边框粗细、边框和填充的颜色

    Rectangle 按钮，按下该按钮后即可进行矩形的绘制

    画椭圆

    使用绘图工具在画布上绘制矩形，以鼠标按下点作为圆外切矩形的左上角点，并可以确定矩形的边框粗细、边框和填充的颜色

    Ellipse 按钮，按下该按钮后即可进行椭圆的绘制。 如果绘制过程中按下 shift 键，则可绘制圆型

    画直线

    使用鼠标在画布上绘制直线，以鼠标的落下点和第二个落下点作为直线的起始点和终结点

    Line 按钮，按下该按钮后即可进行直线的绘制。

    尽量和 Macromedia flash 中的直线行为一致

    使用铅笔工具画线

    使用鼠标在画布上绘制笔迹，如果鼠标按下，则在相应的地方出现墨迹，否则不出现。

    绘制完成之后，自动对所绘轨迹进行变化，可以有向直线、曲线和不变等各种变化方式

    Pencil 按钮，按下该按钮后，若按下鼠标左键则可进行绘制。

    钢笔工具

    使用拖拽方式绘制曲线，使用曲线的控制点来控制曲线的弯曲程度

    Pen 按钮，按下该按钮后，鼠标在舞台上点下的第一个点是笔迹的起始点，接下来的一个点是第二点，按住第二点不松，拖拽得到控制点的位置，也以此确定曲线的弯曲方向和程度

    选择工具

    选择舞台上的各种图形，并可对图形的位置进行拖拽和摆放。 选中某个图形后，会标识该图形为选中状态，可以对该图形在属性面板中设置其属性。

    Drag 按钮，选中该工具之后可以对舞台上的图形进行选择， 并进行相应的拖拽和设置工作。

    变形工具

    对舞台上的图形进行形状的变换，选中图形后，出现该图形的控制点。可以控制该图形的形状变化，包括挤压、拉伸、扭曲和旋转等

    Transform 按钮，选中该工具后可以对舞台上的图形进行变换

    文本工具

    在舞台上写入文字，可以设置文字的字体、大小等

    Text 按钮，选中该工具后可以在舞台上加入文字

    填充工具

    对舞台上未填充的图形按照选择的样式进行填充，对于已填充的图形，则重新填充

    Fill 按钮，选中该工具后可以对舞台上的图形进行填充

    刷子工具

    绘制有轻有重的笔迹，就像用刷子绘制的一样

    Brush 按钮

    动画工具

    生成舞台上多个层上的关键帧之间的过渡帧，即生成动画

    Animation 按钮，现在是通过该按钮生成动画，但是应该在不久的将来使用快捷键或其他的方法来生成

    导出工具

    根据舞台上的动画导出 swf 文件，力求做到和 Eclipse 中生成的动画行为一致

    Export 按钮，不久以后应该整合到菜单中去。

    2.2. workarea 绘画的舞台，可以用时间轴控制其层和帧相关的操作。每一帧的各个层能够在舞台上同时编辑，也可以通过属性面板控制舞台的大小。

    2.3. workarea1 当点Animation按钮时，在workarea1中显示出动画效果，此时点Export按钮可以导出swf文件。（目前在windows下编译存在问题）考虑是否完全仿照macromedia flash的做法，即预览时就是生成swf文件，而在workarea1中显示的动画只是按下Enter键之后在workarea中显示。

    2.4. TimeLine(TimeAxis)

    功能名称

    功能详细描述

    相关操作描述

    增加图层

    增加一个图层（生成图层的 index 为已存在的图层个数减一，而显示出来的效果是 LayerX ，其中 X 是比当前最大的 Layer 数大一，这与 Flash 中是一致的）

    点击 insert 按钮增加一个图层

    删除图层

    删除一个图层（当前仅有一个图层时将不可删除）

    点击 delete 按钮删除选定的图层

    列表框

    显示已存在的各个图层

    点击某个图层使这个图层处于选中状态，继而对这个图层进行相应操作

    时间轴刻度尺

    未显示刻度数， 目前以 toolTipText 实现显示选中的刻度数，并实现插入关键帧操作

    拖拽刻度尺至某个时刻，单击右键选择 insert a keyframe 以增加一个关键帧，然后在 workarea 面板里绘制这个新增的关键帧

    时间轴表格

    目前未实现功能， 预期实现每个表格都能单击以弹出菜单选择增加关键帧，增加帧等操作，单击任意一个表格使这个层处于激活状态

    在某个表格单击右键，选择相应操作

    时间轴。 可以增加和删除图层，选中某一层后可以插入关键帧（未实现插入帧与选中某一层的一个小格激活状态）

    2.5. Property 属性栏。 暂分为两个group，上面的group主要起显示作用，当选中Rectangle按钮时，setText(“rectangle”)，并在下面显示一个矩形图片。 下面的group有种状态 1．当点Rectangle或其他按钮准备画图时，显示两个可供选择颜色的按钮，第一个为笔轴选择，第二个为填充选择。 2．当图形画完后，点drag（或在以后新增一个选中按钮）按钮时显示其长，宽，笔轴颜色及粗细，填充颜色等可供修改。
