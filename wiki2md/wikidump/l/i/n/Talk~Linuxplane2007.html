参考文献的问题

关于第一阶段的工作的若干问题基本理论方面，可以参考《操作系统设计与实现》一书。体系结构相关的部分，请大家给出一些参考文献。目前我知道的内核书籍里面只有一本书讲到了PowerPC体系结构的，书名是《.Linux.Kernel.Primer.A.Top.Down.Approach.for.x86.and.PowerPC.Architectures》,锦波看一下能否提供一些arm体系相关的文献。内核相关的参考文献，我认为还应该给出一些，例如《Linux Kernel Development》。

以下是一些参考书籍

  1. 《Linux内核情景分析》 第十章 系统引导和初始化
  2. 《深入理解Linux内核》 附录 A System Startup
  3. 《Linux Kernel Internals》 第一章 Booting
  4. 《The Linux® Kernel Primer: A Top-Down Approach for x86 and PowerPC Architectures》 第八章 Booting The Kernel
  5. 《Linux内核完全剖析》第一章至第六章
  6. 《自己动手写操作系统》 第一章至第四章

更多的参考论文和手册，可以访问@46\Document\Kernel\Boot ,已经由于墨等同学做了一定程度的整理。大家在研究过程中发现更好的参考资料，请上传到该文件夹中，并发信通知大家。

参考网站：（更多的网站希望大家补充）

  1. Linux Journal
  2. Kernel Trap
  3. Linux Document Project(China Mirror)
