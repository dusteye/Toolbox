简述

本书由内核大牛Robert Love编写，尝试从底层观点来分析Linux系统编程，对于那些致力于要写出高效得代码的人应该会有所帮助。

[编辑]

目标

  • 熟悉Linux下系统编程，例如文件、进程、信号、定时器、socket等编程知识
  • 熟悉Linux下常用开发工具
  • 掌握分析较大项目源代码的能力
