代码库基本信息

大家在日常开发中经常会测试一些小的功能，实现一些小的代码片段。为了有效地在实验室内部实现有效地知识共享，我在SVN上设立了代码库，大家可以将日常完成的代码片段提交到代码库。

  1. 收录项目种类：各类代码均可提交(建议将部分管理脚本也收入代码库)
  2. 代码库地址：svn://svn.footoo.org/opt/codelib/ ，大家可以通过该url访问所有的代码库
  3. 项目名称: svn://svn.footoo.org/opt/codelib/your_code_project
  4. 访问权限：用户与密码设置与之前的相同
[编辑]

示例代码提交注意事项

  1. package name 一律使用：org.footoo.*
  2. 提交前，请将代码提交给吴晋进行审查，由吴晋统一提交
  3. 提交的代码尽量简短，注释清晰，最好在注释里提供有参考价值的URL
[编辑]

现有代码库列表

  1. ShareViaIntent: 使用Android系统的intent机制，将文字、图片分享到其他应用(Gmail,社交网站等)
  2. GrayCapture: Android平台上的即时图像处理框架（不包括图像处理算法），实现提取Camera Preview中的一帧，并保存为灰度图像。在Preview过程中每隔1500ms自动对焦一次，以保证Preview图片质量。代码大部分提取自ZXing，略有修改。
  3. ThreadTest:Java中的线程测试代码，主要是测试Thread.jion函数的用法。
  4. ImageButtonExample: ImageButton的测试代码，根据button状态自动切换图片。
