调查问卷讨论


基本信息

  1. 学历 性别 研究方向 -- 基本信息 用于用户分类
  2. 做过几年研究工作 -- 研究的经验 用于分析调查材料的价值权重合用户分类
  3. 发表、撰写的论文数量

心理状态

  1. 是否经常激励自己做研究的信心
  2. 郁闷时采取什么方式缓解
  3. 是否常常感到研究中走错了方向
  4. 是否曾在研究过程中感到失去方向，如果有又是如何解决的

研究应用

  1. 常用的研究辅助软件（应该给出列表，统计频度） -- 了结常用哪类工具
  2. 平时使用的软件中有那些不便
  3. 本专业领域的权威网站
  4. 这个网站对你最有用的是哪些服务？
  5. 科研资料整理 使用的工具
  6. 对在线写文档的兴趣
  7. 感觉这个工具的优缺点
  8. 希望是怎样的服务 记录方式 整理方式（自动 手动 半自动等）

论文

  1. 最重要的论文怎么得到
  2. 论文完成一般需要到长时间
  3. 附录添加的功能相关问题
  4. 维护参考文献时候遇到哪些不便，希望以何种形式完成维护工作

会议

  1. 你是否关注本领域内的会议及会议报告
  2. 关注那些会议
  3. 关注的原因
  4. 获得会议通知和邀请的渠道又哪些

休闲

  1. 每天/周用于休闲的时间
  2. 休闲的方式
  3. 是否经常泡ｂｂｓ耽误了研究的工作

合作&交流

  1. 相互交流的主要途径
  2. 是否经常和研究组以外的本领域人员交流 采取什么方式？
  3. 是否会和其他领域的学者合作
  4. 合作中遇到的问题
  5. 一个研究团队通常是怎样把握研究进度的
  6. 团体的合作中经常遇到的问题
  7. 一个研究团队通常是怎样分工的
  8. 研究组织内有没有自己的讨论平台
  9. 如果有是什么形式ｂｂｓ　ｗｉｋｉ　ｂｌｏｇ？
  10. 及时交流工具是什么 qq msn gtalk ……
  11. 非及时交流工具是什么 Ｅｍａｉｌ　电话　。。。
  12. 人际交往一般是什么方式 网上交流 还是 当面交流
  13. 组内非面对面讨论的形式 使用工具
  14. 这种方式的优缺点

共享

  1. 会不会把自己的研究资料上传共享 a 上传并有选择的共享 b上传并全部共享 c 上传但不共享 d不上传
  2. 会不会把自己的研究资料上传供自己使用 a 上传 b不上传 因为上传麻烦 c不上传 因为不安全 d不上传 不喜欢

资料整理积累

  1. 日常工作记录的形式
  2. 是否经常记录自己的想法
  3. 通过什么方式记录
  4. 自己的资料以什么方式整理

日志blog

  1. 是否经常关注自己领域的blog
  2. 是否有自己的blog A 有 B 没有 （选B的话不用回答下面的问题）
  3. 访问自己blog的频率（包含非更新登录） A 每周少于7次 B 每周7到20次 C 多于每周20次
  4. 每周blog更新的文章数量 A 3篇一下 B 3~8篇 C 8~15篇 D 15篇以上
