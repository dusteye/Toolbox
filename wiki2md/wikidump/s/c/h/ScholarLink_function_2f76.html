ScholarLink function

Wikipedia，自由的百科全书


1.链接到用户的blog。

2.存放即时消息，留言以及回复等。

3.收藏夹的内容可以设置为private和public。

4.提供一些有用的小工具，进行个人事务的处理。例如个人信息的设置，RSS阅读器，计划约会的提醒等。

5.可供用户个性化的添加自己关注的网页或blog，并用RSS实现内容的更新。

6.记录个人基本信息，做出的贡献和别人的评价。

7.根据选择左侧不同的选项卡显示不同的内容。

8.显示用户的朋友圈，用户可以看到朋友的朋友们，扩展自己的交流圈。

9.显示用户所加入的组织。

10.根据选择左侧不同的选项卡显示内容，用于公告的公布，问题的讨论等。

11.想法来自于HOT，专贴有关技术的文章。

12.供多人协作完成同一份文档，记录修改的时间，内容和作者，提供版本控制的服务。

13.供多人在线同时完成同一张思维导图，应该算是一项可能的创新点。

14.收录个人收藏夹中设置为public的书签，利用tag进行分类，在用户用搜索引擎进行搜索时显示收藏夹中的相关内容。

15.给用户提供便捷的列表服务，其内容可以相当广泛，可以是书籍的列表，会议记录的列表等等。列表中的每项内容可以设置许多相关的属性，例如是书籍的列表，就可以加入书籍的浏览次数、评论、是不是已经借出。

16.按学科分类的目录，利用群体智慧更新的论文集（快速地寻找相关问题的信息和资源，为新进入某一领域或想了解某一领域的人最快的掌握最有效的信息提供方便）。

17.根据基本条件和一系列优化条件搜索志同道合的人。
