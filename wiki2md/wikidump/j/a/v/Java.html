推荐书目

  • 下面推荐的文章中所提到的O'Reilly的书籍，包括：

《Java in a Nutshell》

《Java Examples in a Nutshell 》

《Java Threads, 2nd Ed.》

《Database Programming with JDBC and Java, 2nd Edtion 》

《Java Swing 》

《Java 2D Graphics 》

《Java Virtual Machine》

《Java Internationalization 》

《Java Enterprise in a Nutshell 》

…… 当然，看书要一步一步的来，心急吃不了热豆腐。而且有些书籍中的内容并非经常用到（不过用到的时候去查会发现其妙处）。所以，推荐以下入门书籍：

  • 《Core Java》I、II两册，作为入门，看第一册就基本足够
  • 《Java 2 编程指南》 比较不错的一本书，作为Java的入门书籍可以算及格吧
  • 《Java编程思想》非常经典，在某些主题上的深度也很足够。
  • 《Java2全方位学习》
